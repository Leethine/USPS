# USPS
United Scheme Preprocessor System

## About
This project aim to create a preprocessor to unify various scheme implementations as much as possible.

## How does it work
Using shebang and C-style macros, the system will decide which scheme implementation is being used and what code to replace in case of the implementation selected. A set of pre-processed scm files will be generated.
